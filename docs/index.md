# Bem vindo ao Projeto FMS

### [Site do projeto](https://fmswebscrapping.com.br/login)

O projeto **Fms WebScrapping** tem como premissa a coleta de dados de E-coomerces como o **Mercado Livre** ou **Pneu Store** para acompanhamento de produtos que são de interesse comum, acompanhando variações de preços em diversas lojas disponíveis dentro do Mercado Livre ou Amazon, que trabalham com lojas externas ou Lojas individuais.

Com isso, posso analisar variações e capturar os melhores produtos com o objetivo de economizar, não só para mim, mas para qualquer um que queira acompanhar produtos em uma determinada janela de tempo.

# ✨ Um problema de negócio?!
O projeto pode resolver problemas de negócio como pesquisa de mercado e concorrência ou análise de disponibilidade de produtos disponibilizados em comercios como iFood ou vitrines disponibilizadas diretamente no search do google.

## 🚀 Recursos Principais
    - Coleta de dados
    - Mapeamento de tabelas
    - Conecção com Banco de Dados

## 🛠️ Tecnologias
    - Mysql
    - Python
    - Pandas
    - SqlAlchemy (ORM)
        - O projeto conta com a modelagem de tabelas orquestrada com SqlAlchemy ORM (**Object Relational Mapping**) que tem como responsabilidade transformar classes Python em tabelas de um determinado banco de dados. 

    - Requests
    - Beatifousoup
    - Airflow
    - Mkdocs - Material
    - Self Host: Hostinger
        - Todo o projeto está hospedado em uma VPS da hostinger, tendo 100% de disponibilidade e autonomia.
